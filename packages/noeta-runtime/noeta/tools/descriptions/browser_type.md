Types `text` into the input element at `index` in the sandbox browser (replacing any existing value) and returns the result; set `submit` to true to press Enter after typing (e.g. to submit a search box).

The `index` is the number of the element (e.g. `[3]`) from the numbered list a prior `browser_navigate` or `browser_extract` returned — read the page first, then type into the element by index. Call `browser_extract` afterward to see the resulting page.
